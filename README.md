 # 🚀 Space Explorer Hub

A responsive and interactive web application that allows users to explore space-related content while also providing real-time weather updates and a personal to-do manager.

## 📌 Features

- 🌌 Explore planets, galaxies, rockets, and astronauts
- 🌦️ Real-time Weather App using Weather API
- ✅ To-Do List with Local Storage
- 🌙 Dark/Light Mode
- 📱 Fully Responsive Design
- ⚡ Optimized Performance
- 🔍 SEO Optimized

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Local Storage
- Git & GitHub
- GitHub Pages

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/archana22206/space-explorer-website.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
